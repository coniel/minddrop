- always use block for `if`
- empty lines above blocks and return
- no hard-coded language strings (allowed in dev tools), always use i18n and add default translation to
  packages/i18n/src/locales/en-GB.json
- use ui/primitives/src/ScrollArea/ScrollArea.tsx for all scrollable content
- class names: `button-color-red` not `btn-color--red`
- var names: use whole words, not abbreviations (e.g. use `property` not `prop`, `event` not `e`)
- icon set: lucide react
- after adding a @minddrop workspace package as a dependency, run `pnpm i` in the workspace package
- prefer using Group and Stack components over divs with css for layout
- use var(--space-#) for spacing instead direct numbers
